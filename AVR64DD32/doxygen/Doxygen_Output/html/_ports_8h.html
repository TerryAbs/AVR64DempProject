Detailed Description

GPIO Port driver for AVR64DD32.

Implements GPIO port configuration and control. Supports digital input/output, pull-up resistors, interrupt-on-change, and pin inversion. Provides high-level interface for port manipulation.

Author
Terry (ABS Data Systems)
Date
November 25, 2025
Version
1.0
Available Ports
  • PORT A (PA0-PA7): I2C, SPI, USART
  • PORT C (PC0-PC3): USART, ZCD
  • PORT D (PD0-PD7): ADC, DAC, CCL outputs, Event outputs
  • PORT F (PF0-PF6): CCL inputs, Timer output, ZCD simulator
Pin Features
  • Direction control (input/output)
  • Pull-up resistor enable
  • Input level configuration
  • Pin inversion
  • Interrupt sense control
Demo Pins
  • PC3 (TP16): General purpose I/O demo
  • PD1-PD3: PORT/CCL test outputs
  • PF6 (H2-11): Input-only test pin
Note
Some pins have input-only capability (PF6 on 32-pin package)
