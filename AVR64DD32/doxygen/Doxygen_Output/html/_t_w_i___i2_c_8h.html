Detailed Description

Two-Wire Interface (I2C) driver for AVR64DD32.

Implements I2C master mode communication with support for EEPROM (24LC512) and FRAM (FM24C512C) devices. Provides read, write, and addressing functions with automatic ACK/NACK handling.

Author
Terry (ABS Data Systems)
Date
November 25, 2025
Version
1.0
Pin Configuration
  • PA0: SDA (Serial Data) - requires pull-up resistor
  • PA1: SCL (Serial Clock) - requires pull-up resistor
Supported Devices
  • Microchip 24LC512 EEPROM (512Kbit, 0x50-0x57)
  • Cypress FM24C512C FRAM (512Kbit, 0x50-0x57)
Features
  • 7-bit addressing
  • Configurable bus speed (100kHz/400kHz)
  • Page write support
  • Random read access
  • Bus recovery
Note
External pull-up resistors required (2.2k? - 4.7k? typical)
EEPROM requires write delay (~5ms), FRAM does not
Warning
No clock stretching support - ensure slave device compliance
Bus arbitration not implemented for multi-master scenarios
