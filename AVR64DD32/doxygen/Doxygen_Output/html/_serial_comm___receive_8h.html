Detailed Description

Serial command reception and parsing for AVR64DD32 system.

Implements buffered reception of serial commands from USART1. Handles line editing (backspace), echo, and command termination (Enter key). Integrates with menu processor for command execution.

Author
Terry (ABS Data Systems)
Date
November 25, 2025
Version
1.0
Command Processing
  1. Buffer received characters
  2. Echo characters to terminal
  3. Handle backspace/delete
  4. Detect Enter (newline)
  5. Pass command to Menu Processor
  6. Clear buffer for next command
Features
  • Line editing with backspace support
  • Command echo to terminal
  • Case-insensitive command matching
  • Command prompt display
  • Buffer overflow protection
Buffer Management
  • Circular buffer for received characters
  • Automatic overflow handling
  • Configurable buffer size
Note
Integrates with USART interrupt handler
Commands are whitespace-terminated
Warning
Buffer size limits maximum command length
