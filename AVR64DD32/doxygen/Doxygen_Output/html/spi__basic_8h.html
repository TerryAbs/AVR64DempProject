enum spi_transfer_status

Status of the SPI hardware and SPI bus.

Enumerator
SPI_FREE 

SPI hardware is not open, bus is free.

SPI_IDLE 

SPI hardware has been opened, no transfer ongoing.

SPI_BUSY 

SPI hardware has been opened, transfer ongoing.

SPI_DONE 

SPI hardware has been opened, transfer complete.

◆ spi_transfer_type

enum spi_transfer_type

Specify whether the SPI transfer is to be uni- or bidirectional. A bidirectional transfer (=SPI_EXCHANGE) causes the received data to overwrite the buffer with the data to transmit.

Enumerator
SPI_EXCHANGE 

SPI transfer is bidirectional.

SPI_READ 

SPI transfer reads, writes only 0s.

SPI_WRITE 

SPI transfer writes, discards read data.
