Detailed Description

EEPROM memory map definitions for AVR64DD32 project.

Defines address allocations for peripheral configuration storage in internal EEPROM. Organizes memory by module with defined boundaries to prevent overlap.

Author
Terry (ABS Data Systems)
Date
November 25, 2025
Version

Memory Organization
Total: 512 bytes (0x0000 - 0x01FF)
