Detailed Description

Zero-Cross Detector driver for AVR64DD32.

Implements ZCD (Zero-Cross Detector) for detecting AC zero-crossing points. Useful for AC phase control, dimmer circuits, and AC frequency measurement. Can generate interrupts on positive or negative zero crossings.

Author
Terry (ABS Data Systems)
Date
November 25, 2025
Version
1.0
Pin Configuration
  • PC2 (TP14): ZCD Input (AC signal)
  • PA7: ZCD Output (optional)
  • PF5 (TP5): ZCD Simulator (test signal generator)
Features
  • Detects positive and negative zero crossings
  • Interrupt generation on crossing events
  • Output signal on zero crossing
  • Software-enabled/disabled
Typical Applications
  • AC dimmer control (TRIAC triggering)
  • AC frequency measurement
  • Power line synchronization
  • AC phase control
Test Circuit
Use PF5 to generate test waveform for ZCD input on PC2. Resistor divider may be needed to scale AC voltage to safe levels.
Note
Input voltage must be within safe operating range (see datasheet)
Warning
Do NOT connect mains AC directly - use isolation transformer
Ensure proper voltage scaling and isolation
ZCD has specific input voltage requirements (0-3.3V range)
