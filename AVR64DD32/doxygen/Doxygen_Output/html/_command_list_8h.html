Detailed Description

Command parser and dispatcher for AVR64DD32 system.

Implements command table lookup and execution. Maps text commands to function callbacks. Supports command history and help system.

Author
Terry (ABS Data Systems)

Command Structure
Each command contains:
  • Command string (e.g., "adc", "dac")
  • Description (help text)
  • Function pointer (callback on match)
Command Processing
  1. Receive command string
  2. Search command table
  3. Execute callback if found
  4. Display error if not found
Built-in Commands
  • help: Display command list
  • clear: Clear screen
  • version: Show firmware version
  • reset: Software reset
Note
Commands stored in PROGMEM
