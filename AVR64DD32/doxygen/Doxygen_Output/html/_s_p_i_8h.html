Detailed Description

Serial Peripheral Interface driver for AVR64DD32.

Implements SPI master mode with support for multiple flash memory devices including W25Q128, AT25SF641, and SST26VF016B. Provides read, write, and erase operations with device-specific commands.

Author
Terry (ABS Data Systems)
Date
November 25, 2025
Version
1.0
Pin Configuration
  • PA4: MOSI (Master Out Slave In)
  • PA5: MISO (Master In Slave Out)
  • PA6: SCK (Serial Clock)
  • PA7: SS (Slave Select)
Supported Flash Devices
  • Winbond W25Q128 (128Mbit)
  • Adesto AT25SF641 (64Mbit)
  • Microchip SST26VF016B (16Mbit)
Operations
  • Device ID read and verification
  • Page write (up to 256 bytes)
  • Sector/block erase
  • Sequential read
  • Status register access
Note
Uses interrupt-driven transfers for efficiency
Automatic busy polling during erase/write operations
Warning
Ensure proper SPI mode (MODE 0 or MODE 3) for your device
SS must remain low during entire transfer
Flash write operations require prior erase
