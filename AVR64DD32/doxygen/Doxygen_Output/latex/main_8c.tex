\doxysubsection{Detailed Description}
Main entry point for AVR64\+DD32 educational demonstration project. 

Comprehensive demonstration of AVR64\+DD32 micro controller peripheral features including ADC, DAC, USART, SPI, I2C, timers, events, and more. Implements a VT102 terminal-\/based menu system for interactive hardware exploration and configuration.
