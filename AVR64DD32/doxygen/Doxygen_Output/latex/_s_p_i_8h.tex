\doxysubsection{Detailed Description}
Serial Peripheral Interface driver for AVR64\+DD32. 

Implements SPI master mode with support for multiple flash memory devices including W25\+Q128, AT25\+SF641, and SST26\+VF016B. Provides read, write, and erase operations with device-\/specific commands.

\begin{DoxyParagraph}{Pin Configuration}

\begin{DoxyItemize}
\item PA4:\+ MOSI (Master Out Slave In)
\item PA5:\+ MISO (Master In Slave Out)
\item PA6:\+ SCK (Serial Clock)
\item PA7:\+ SS (Slave Select)
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Supported Flash Devices}

\begin{DoxyItemize}
\item Winbond W25\+Q128 (128Mbit)
\item Adesto AT25\+SF641 (64Mbit)
\item Microchip SST26\+VF016B (16Mbit)
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Operations}

\begin{DoxyItemize}
\item Device ID read and verification
\item Page write (up to 256 bytes)
\item Sector/\+block erase
\item Sequential read
\item Status register access
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyNote}{Note}
Uses interrupt-\/driven transfers for efficiency 

Automatic busy polling during erase/\+write operations 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Ensure proper SPI mode (MODE 0 or MODE 3) for your device 

SS must remain low during entire transfer 

Flash write operations require prior erase
\end{DoxyWarning}
