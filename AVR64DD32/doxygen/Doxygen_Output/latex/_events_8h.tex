\doxysubsection{Detailed Description}
Event System driver for AVR64\+DD32. 

Implements the Event System (EVSYS) for hardware-\/based peripheral interconnection without CPU intervention. Allows direct routing of events between peripherals (timers, ADC, CCL, ports, etc.).

\item 10 event channels (0-\/9)

\item Direct peripheral-\/to-\/peripheral connection
\item Software event generation
\item Port pin event inputs

\item Port pin change ? CCL input
