\doxysubsection{Detailed Description}
Real-\/time data display processor for AVR64\+DD32 system. 

Manages automatic display of peripheral data in the terminal\textquotesingle{}s right-\/side data area (columns 85-\/120). Supports registration of display items with custom formatting callbacks.

\item Custom formatting function

\item Enable/\+disable flag

\begin{DoxyItemize}
\item Location:\+ Columns 85-\/120, rows 1-\/30
\item Update:\+ Automatic on each main loop iteration

\end{DoxyParagraph}

\item Custom formatting via function pointers
