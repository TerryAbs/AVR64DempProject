\doxysubsection{Detailed Description}
Color theme definitions for terminal interface. 

Provides predefined color schemes for consistent visual appearance across menus and data displays. Themes specify foreground and background color combinations.

\begin{DoxyParagraph}{Predefined Themes}

\begin{DoxyItemize}
\item Header theme:\+ Menu titles and section headers
\item Body theme:\+ Menu items and descriptions
\item Data theme:\+ Real-\/time data display
\item Prompt theme:\+ User input areas
\item Error theme:\+ Error messages and warnings
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Theme Structure}
Each theme contains:\+
\begin{DoxyItemize}
\item Foreground color (text)
\item Background color (behind text)
\end{DoxyItemize}
\end{DoxyParagraph}
