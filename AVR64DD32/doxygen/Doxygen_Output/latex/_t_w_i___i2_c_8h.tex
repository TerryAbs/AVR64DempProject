\doxysubsection{Detailed Description}
Two-\/\+Wire Interface (I2C) driver for AVR64\+DD32. 

Implements I2C master mode communication with support for EEPROM (24LC512) and FRAM (FM24\+C512C) devices. Provides read, write, and addressing functions with automatic ACK/\+\+NACK handling.

\item PA1:\+ SCL (Serial Clock) -\/ requires pull-\/up resistor

\item Cypress FM24\+C512C FRAM (512Kbit, 0x50-\/0x57)
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Features}

\begin{DoxyItemize}
\item 7-\/bit addressing
\item Configurable bus speed (100k\+Hz/\+400k\+Hz)
\item Page write support
\item Random read access
\item Bus recovery
\end{DoxyItemize}
\end{DoxyParagraph}
