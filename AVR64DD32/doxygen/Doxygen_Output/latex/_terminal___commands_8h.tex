\doxysubsection{Detailed Description}
VT102 terminal control command library. 

Provides functions for VT102/\+\+ANSI terminal control including cursor positioning, screen clearing, color setting, and attribute control. Enables creation of sophisticated terminal-\/based user interfaces.

Terry (ABS Data Systems)

\end{DoxyVersion}
\begin{DoxyParagraph}{Supported Terminal Commands}

\begin{DoxyItemize}
\item Cursor positioning (absolute and relative)
\item Screen clearing (full, partial, line)
\item Foreground and background colors (16 colors)
\item Text attributes (bold, underline, reverse)
\item Cursor visibility control
\item Screen save/\+restore
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Color Palette}

\begin{DoxyItemize}
\item Black, Red, Green, Yellow, Blue, Magenta, Cyan, White
\item Bright variants available
\end{DoxyItemize}
\end{DoxyParagraph}
