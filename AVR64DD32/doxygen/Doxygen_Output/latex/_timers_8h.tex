\doxysubsection{Detailed Description}
Timer/\+\+Counter driver interface for AVR64\+DD32. 

Implements TCB (Timer/\+\+Counter B) modules for PWM generation, frequency measurement, and periodic interrupts. Supports multiple timer modes including periodic interrupt and pulse width modulation.

\begin{DoxyParagraph}{Features}

\item Configurable prescaler (DIV1, DIV2, DIV4, DIV8, DIV16)
\item Interrupt support for overflow and compare match
\item Synchronized timer start/\+stop
