\doxysubsection{Detailed Description}
Zero-\/\+Cross Detector driver for AVR64\+DD32. 

Implements ZCD (Zero-\/\+Cross Detector) for detecting AC zero-\/crossing points. Useful for AC phase control, dimmer circuits, and AC frequency measurement. Can generate interrupts on positive or negative zero crossings.
