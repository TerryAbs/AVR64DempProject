\doxysubsection{Detailed Description}
Implementation of ADC driver for AVR64\+DD32. 

Implements three ADC channels with distinct characteristics:\+
\begin{DoxyItemize}
\item Instant:\+ Single-\/shot reading, fast response
\item Average:\+ Hardware-\/averaged (16 samples), noise reduction
\item Differential:\+ Measures voltage difference between PD5 and PD4
\end{DoxyItemize}

\begin{DoxyAuthor}{Author}
Terry (ABS Data Systems) 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
Created:\+ 11/\+25/\+2025 2:\+01:\+07 PM 
\end{DoxyDate}
\begin{DoxyVersion}{Version}
1.\+0
\end{DoxyVersion}
\begin{DoxyParagraph}{Algorithm}

\begin{DoxyEnumerate}
\item Initialize:\+ Configure VREF, prescaler, sample count
\item Read:\+ Trigger conversion and wait for RESRDY flag
\item Convert:\+ Apply scaling based on reference and resolution
\item Display:\+ Format and output via display processor
\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{State Machine}
IDLE ? INITIALIZE ? READ ? DISPLAY ? COMPLETE ? IDLE
\end{DoxyParagraph}
\begin{DoxyNote}{Note}
Uses function pointers for polymorphic channel behavior 

Results are automatically displayed when channels are enabled 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Ensure global interrupts are enabled for proper operation 

Sample delay and prescaler affect conversion time and accuracy
\end{DoxyWarning}
